## Setup

To run locally:

1. Install Bun >= `1.2`
1. Clone the repo
1. Run `bun i`

You can now run these npm scripts
- `bun run --bun dev` - Start the project with Greenwood's local dev server
- `bun run --bun serve` - Start the project with a production Greenwood build
